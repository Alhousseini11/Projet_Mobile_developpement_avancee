# Admin Web

Frontend web autonome pour l'administration du garage.

## Demarrage

```bash
npm install
npm run dev
```

Par defaut, l'application appelle `http://localhost:3000`.

Pour changer l'API cible, cree un fichier `.env.local` :

```bash
VITE_API_URL=http://127.0.0.1:3000
```

## Build

```bash
npm run build
```
